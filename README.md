# Grade Management System

A Java console-based application to manage student grades using arrays and control flow.

## Features
- Add student marks
- View all students
- Calculate averages and grades
- Generate performance reports
- Top performers
- Grade distribution

## Project Structure
- src/ → Java source files
- output/ → Sample input data

## How to Run
1. Compile:
   javac src/*.java
2. Run:
   java -cp src GradeManagementSystem
